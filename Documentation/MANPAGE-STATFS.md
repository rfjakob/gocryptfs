NAME
====

statfs - dump the statfs information for PATH to console in JSON format

SYNOPSIS
========

#### Examine encrypted file/directory
statfs PATH

DESCRIPTION
===========

There are no options to this command.

EXAMPLES
========

Examine a directory entry:

	statfs myfs/mCXnISiv7nEmyc0glGuhTQ
